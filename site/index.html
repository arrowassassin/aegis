> what is it

AI coding agents now run real commands on real machines — rm -rf, git push --force, terraform destroy, drop table — faster than you can read a confirmation. Aegis sits on that one chokepoint: the command line. It warns in plain English before execution, holds the dangerous ones for a one-key decision, makes destructive actions reversible, and keeps a tamper-evident record of everything every agent did.

Security spine. Rules block; the model only explains. The decision to hold or deny a catastrophic command is made by deterministic rules — never by an LLM, so it can't be argued past by prompt injection. The raw command is always shown verbatim. The log is append-only and hash-chained.

Agent-agnostic from day one: Claude Code, Qwen, Codex, Gemini, or any custom/MCP agent — via a native hook, an MCP server, or a $PATH shim.

> how it works

Three interception points normalize to one event. A resident daemon decides in microseconds and records everything.

AGENT

proposes a command

▶

INTERCEPT

hook · MCP · $PATH shim → one ProposedCommand

▶

DECIDE

Tier-1 rules → SAFE / CATASTROPHIC / AMBIGUOUS; Tier-2 model scores the middle

▶

SNAPSHOT + LOG

reversible + hash-chained audit

SAFE auto-allows on a model-free fast path (~microseconds). CATASTROPHIC is a hard floor. AMBIGUOUS is held for you, or scored against a threshold when you're away.

> features & flow

Every frame below is real output captured from the binaries.

1 · The gate — held before it runs

Catastrophic and ambiguous commands pause with a calm "hold card": one plain-English line, the raw command, and three keys. [a]llow once · [d]eny · [r] always-allow-here.

PATH=…/shims rm -rf src
────────────────────────────────────────────────────────────
⚠ Aegis hold — This command is catastrophic and cannot be undone.
  recursively deletes files and directories. This is hard or impossible to undo.

    rm -rf src

  [a] allow once   [d] deny   [r] always allow here
────────────────────────────────────────────────────────────

2 · The timeline — one cross-agent log

Every command from every wired agent, on a tamper-evident, hash-chained record. Calm by default; the one danger accent appears only when it must.

aegis log
time      agent         outcome  command
02:31:27  shim          held     [catastrophic] rm -rf src
02:31:27  shim          denied   [catastrophic] rm -rf src

3 · The TUI — live, interactive

A real ratatui app over the live log: j/k to move, enter for detail, a/d to approve/deny a held action, u to undo, / to filter. Reflows to any size; restores the terminal on exit.

4 · The approval queue — agents proceed

Held commands are queued, not dropped. Approve from the CLI, the TUI, or let the agent wait in-band — once you approve, the command runs and the result returns, so the agent keeps going.

aegis queue
id          class          command
50d56fd9    catastrophic   rm -rf src

Approve with `aegis approve <id>` or deny with `aegis deny <id>`.

5 · Undo — nothing is unrecoverable

Before an allowed destructive op, Aegis snapshots the paths it will touch (reflink copy-on-write, plain-copy fallback). aegis undo restores the last action; --session restores the whole run.

aegis undo
✓ undid `rm -rf src` (1 path(s) restored)

Restored 1 snapshot(s). Note: undo covers files only — not network
calls, external APIs, or already-pushed commits.

> install

Get the binaries, then wire your agent. One command does both.

cargo install aegis        # or: brew install aegis  (when published)
aegis init                 # detect agents, wire interception, start the daemon

Then run a command through any wired agent — or a raw shell with the shim dir on your PATH — and watch aegis log / aegis tui.

  • Sub-second warnings; SAFE commands add negligible overhead.
  • Fail-open by default (a down daemon never blocks you); AEGIS_FAIL_CLOSED=1 to flip.
  • Panic kill-switch: aegis panic halts everything, aegis resume clears it.

> claude code plugin

Aegis ships as a Claude Code plugin + marketplace. The plugin is a thin wiring layer; the binaries come from cargo/Homebrew.

/plugin marketplace add arrowassassin/aegis
/plugin install aegis@aegis

It registers a PreToolUse hook (→ aegis-hook) and the aegis-exec MCP server (→ aegis-mcp). Ships disabled until you trust it.
